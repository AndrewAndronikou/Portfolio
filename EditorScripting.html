Editor Scripting - Unity C#

​ Editor scripting and tools programming is a new area which I looked into after I finishing my degree, and I wish this is something I looked into prior as the capabilities for aiding game development is immense. As I created bigger projects, I realised that designing levels, characters etc. became cumbersome so by creating tools that would aid the design process this made the whole process much more manageable.​ The enemy class creator is a tool created to assist me with creating various classes, specializations, weapons etc. This utilizes scriptable objects and character data to retrieve the relevant variables for that class, this allowed me to rapidly create prefab enemy classes to use in my games.
